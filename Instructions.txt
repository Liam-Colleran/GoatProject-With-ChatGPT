1. In the command prompt or terminal, navigate to the project directory containing app.py.

2. Run the following command to start the Flask development server: python app.py

3. Open a web browser and access http://localhost:5000 to see the shoe search web page.

4. Enter a search query and the number of shoes in their respective input fields, and click the "Search" button.

5. The results will be displayed below the form in a structured format, including shoe names, images, conditions, release dates, and retail prices.
